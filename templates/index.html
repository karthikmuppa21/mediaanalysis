Need Help?

If you're unsure how to get the YouTube Video ID or API Key, follow these simple steps:

1. How to Get the YouTube Video ID

The YouTube Video ID is a unique identifier for each video on YouTube. Here's how you can find it:

  1. Go to the YouTube video you want to analyze.
  2. Look at the URL of the video. For example, in the URL https://www.youtube.com/watch?v=dQw4w9WgXcQ, the part after ?v= is the Video ID.
  3. In this case, dQw4w9WgXcQ is the Video ID.

2. How to Get Your YouTube API Key

To interact with YouTube's API and fetch data such as video comments, you need an API Key. Here's how you can get it:

  1. Go to the Google Developer Console.
  2. If you don’t have an account, sign up. If you already have one, sign in.
  3. Create a new project by clicking on the Create Project button at the top of the page.
  4. Give your project a name and click Create.
  5. Once your project is created, click on the Navigation Menu on the left side, then go to APIs & Services > Library.
  6. Search for the YouTube Data API v3 and click on it.
  7. Click Enable to enable the YouTube Data API for your project.
  8. After enabling the API, go to Credentials from the left menu.
  9. Click on the Create Credentials button and select API Key.
  10. Your new API Key will appear. You can copy it and use it in the tool.

Now you have both your YouTube Video ID and API Key! You can paste these into the fields above to start analyzing your video comments.
